# คู่มือการซ่อน/แสดง Visual ด้วยเทคนิค Cover-up (Conditional Visibility)

เทคนิคนี้ใช้ตอบโจทย์ด้าน UX/UI เมื่อเราต้องการบังคับให้ User ทำการเลือก Filter (เช่น ต้องเลือกแค่ 1 ปีเท่านั้น) ก่อนถึงจะอนุญาตให้เห็นกราฟได้

หลักการคือ เราจะเอา "แผ่นป้าย (Card Visual)" มาวางทับกราฟที่เราต้องการจะซ่อน แล้วใช้ DAX เปลี่ยนสีแผ่นป้ายให้กลายเป็นสี "โปร่งใสทะลุ" (Transparent) เฉพาะตอนที่ User ทำตามเงื่อนไขที่เราตั้งไว้

---

## 🛠️ Step 1: สร้าง Measure ควบคุมสี (Background Color)
สร้างสูตร DAX นี้เพื่อเช็คว่า User เลือกปีแค่ 1 ปี หรือไม่ โดยใช้ฟังก์ชัน `HASONEVALUE`
```dax
Cover Visibility = 
IF(
    HASONEVALUE('Dim_Date'[Year]), // เงื่อนไข: กดเลือกปีแค่ 1 ปีใช่หรือไม่?
    "#FFFFFF00",  // ถ้าใช่: ให้เปลี่ยนรหัสสีเป็น "โปร่งใส" (รหัส 8 หลัก ลงท้ายด้วย 00)
    "#FFFFFF"   // ถ้าไม่ใช่: ให้เป็น "#FFFFFF" (สีขาวทึบ เพื่อบังให้มิด)
)
```
*(💡 ทริค: ผมให้คุณกลับมาใช้รหัส `#FFFFFF00` เพราะว่าถ้าเราใช้ `BLANK()` Power BI จะดึงสีขาวดั้งเดิมของ Card มาใช้แทน ทำให้มันไม่ยอมโปร่งใสครับ)*

---

## 🛠️ Step 2: สร้าง Measure ควบคุมข้อความเตือน (Text Message)
เพื่อให้แผ่นป้ายที่มาบัง มีข้อความสื่อสารบอก User ด้วยว่าทำไมกราฟถึงโดนบังอยู่ ให้สร้างสูตรนี้ครับ:
```dax
Cover Text = 
IF(
    HASONEVALUE('Dim_Date'[Year]),
    "", // ใช้เครื่องหมายฟันหนูติดกัน (Empty String) เพื่อไม่ให้ Power BI พิมพ์คำว่า (Blank) ออกมา
    "Please select 1 Year to view this chart" // ถ้ายังไม่เลือก หรือเลือกหลายปี ให้ขึ้นข้อความเตือนนี้
)
```

---

## 🛠️ Step 3: ประกอบร่างแผ่นป้ายล่องหน
1. คลิกที่พื้นที่ว่างบน Dashboard แล้วสร้างกราฟ **Card** แบบปกติ
2. ลากฟิลด์ **`Cover Text`** จาก Step 2 มาใส่ใน Card (ตัว Card จะโชว์ข้อความเตือนขึ้นมา)
3. ไปที่แถบ Format ปิด Category label ของ Card ทิ้งไป และจัดข้อความให้อยู่กึ่งกลาง
4. จับ Card อันนี้ ย่อ/ขยาย แล้วลากไป **วางทับ** กราฟที่คุณต้องการจะซ่อน ให้มิดชิดที่สุด (อาจจะต้องคลิกขวาที่ Card เลือก Bring to front เพื่อให้อยู่บนสุด)
5. ไปที่แท็บ **Format (รูปแปรงทาสี) > Background**
6. คลิกปุ่ม **`fx`** (Conditional formatting) ของ Background
   - Format style: เลือกเป็น **Field value**
   - What field should we base this on?: เลือก **`Cover Visibility`** จาก Step 1
   - กด OK

🎉 **ทดสอบผลงาน:** 
ลองไปคลิกที่ Slicer ปี (Year) ดูครับ 
- ถ้าคุณเลือก 1 ปี แผ่นป้ายจะโปร่งใสและหายไป ยอมให้คุณเห็นกราฟด้านล่าง
- แต่ถ้าคุณเลือกหลายปี หรือไม่เลือกเลย แผ่นป้ายจะกลายเป็นสีขาวทึบและมีคำเตือนขึ้นมาบังมิดทันทีครับ!
