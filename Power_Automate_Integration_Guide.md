# คู่มือการฝัง Power Automate ลงใน Power BI (เตรียมสัมภาษณ์ TCS)

> [!WARNING]
> **ข้อจำกัดเรื่อง Account:** ฟีเจอร์การดึง Visual จาก AppSource และการล็อกอินเข้า Power Automate จำเป็นต้องใช้ **บัญชีองค์กร (Work/School Account)** ครับ หากคุณไม่มีบัญชีองค์กร จะไม่สามารถกดเล่นทำตามแบบ Hands-on ได้ 
> 
> **แต่ไม่ต้องกังวลครับ!** สำหรับการเตรียมตัวสัมภาษณ์ คุณไม่จำเป็นต้องเปิดคอมทำให้กรรมการดูสดๆ อยู่แล้ว ขอแค่ **"เข้าใจคอนเซปต์และจำสคริปต์คำตอบ"** ในบรรทัดล่างสุดของเอกสารนี้ไปเล่าให้กรรมการฟัง ก็เพียงพอที่จะทำให้เขาประทับใจแล้วครับ (เมื่อเข้าทำงานจริง ทาง TCS จะมี Account ระดับ Enterprise ให้คุณใช้อยู่แล้วครับ)

นี่คือ **"หมัดฮุก"** สำหรับการสัมภาษณ์ครั้งนี้ครับ เพราะใน Resume ของคุณระบุว่ามีประสบการณ์ด้าน Power Automate มาก่อน 

การที่คุณสามารถโชว์ให้กรรมการเห็นว่าคุณสามารถทำ **Cross-platform Integration** (เชื่อมต่อ Power BI เข้ากับระบบ Automation ได้) จะทำให้โปรไฟล์คุณยกระดับจาก Data Analyst ธรรมดา กลายเป็น **Microsoft Power Platform Developer** แบบเต็มตัวเลยครับ!

---

## 🛠️ Step 1: เพิ่ม Visual "Power Automate for Power BI"
1. เปิดหน้า Dashboard ของคุณใน Power BI Desktop
2. มองไปที่แผง **Visualizations** ทางขวามือ
3. หาไอคอนรูป **Power Automate** (โลโก้ลูกศรสีฟ้า) 
   - *หมายเหตุ: หากคุณใช้ Power BI เวอร์ชันเก่าและหาไอคอนนี้ไม่เจอ ให้คลิกที่จุดสามจุด **`...` (Get more visuals)** ท้ายสุดของแผง Visualizations > เลือก **Get more visuals** > ค้นหาคำว่า "Power Automate" แล้วกดปุ่ม Add ครับ*
4. คลิกที่ไอคอนเพื่อสร้าง Visual นี้ลงบน Dashboard แล้วจัดตำแหน่งปุ่มให้สวยงาม
5. **กติกาสำคัญ:** คุณต้องโยน "Data" เข้าไปให้ตัว Visual ก่อน มันถึงจะยอมให้สร้าง Flow ได้
   - ให้ลากฟิลด์ที่คุณอยากจะให้ส่งข้อมูลไปพร้อมกับอีเมล (เช่น ลากฟิลด์ `[Region]` และ measure `[Total Sales]`) ไปใส่ในช่อง **Power Automate Data** 

---

## 🛠️ Step 2: สร้าง Flow ส่งอีเมล (Send an Email)
1. เมื่อใส่ Data เสร็จแล้ว ให้คลิกที่ปุ่ม **`...` (More options)** บริเวณมุมขวาบนของตัว Visual Power Automate
2. เลือกคำสั่ง **Edit (แก้ไข)** 
   *(รอระบบโหลดสักครู่ หากขึ้นให้ Sign in ให้ใช้อีเมล O365 ของคุณ)*
3. หน้าจอจะเปลี่ยนเข้าสู่ระบบ Power Automate (แต่ยังฝังอยู่ใน Power BI) 
4. คลิกเมนู **+ New** (ด้านบน) > เลือก **Instant cloud flow**
5. ระบบจะมี Trigger มาให้ 1 ตัวตั้งต้นคือ **"On Power BI button clicked"**
6. คลิกปุ่ม **+ New step** ด้านล่าง
7. ค้นหาคำสั่ง **`Send an email (V2)`** (ของ Office 365 Outlook)
8. กำหนดค่าการส่งอีเมลดังนี้:
   - **To:** ใส่อีเมลของคุณเองเพื่อทดสอบ
   - **Subject:** `🚨 Alert: Superstore Sales Report Action Required`
   - **Body:** `เรียนผู้จัดการ, กรุณาตรวจสอบยอดขายบน Dashboard ทันทีครับ` 
   *(💡 ทริค: คุณสามารถคลิก Dynamic Content เพื่อดึงข้อมูล `Region` หรือ `Total Sales` ที่เราลากมาใส่ใน Step 1 มาโชว์ในเนื้อหาอีเมลได้ด้วยครับ!)*
9. กดปุ่ม **Save** (บันทึก) และกดปุ่ม **Save and apply** ด้านบนสุด

---

## 🛠️ Step 3: ปรับแต่งปุ่มและทดสอบยิงอีเมล
1. กดปุ่ม **Back to report** (ลูกศรย้อนกลับที่มุมซ้ายบน) เพื่อกลับมาที่หน้า Dashboard
2. ตอนนี้ Visual ของคุณจะกลายร่างเป็น **"ปุ่มกด (Button)"** แล้ว!
3. ไปที่แถบ **Format (รูปแปรงทาสี)** ของปุ่มนี้
4. ตรงหัวข้อ Button text ให้พิมพ์ข้อความเท่ๆ ลงไป เช่น **"📧 Send Report to Manager"**
5. ปรับเปลี่ยนสีปุ่ม (Fill color) ให้เข้ากับโทนสี Dashboard ของคุณ
6. **วิธีทดสอบ:** ใน Power BI Desktop การจะกดปุ่มได้ คุณต้องกดปุ่ม **`Ctrl` + คลิกซ้าย** บนเมาส์ครับ 
   - พอกดปุ่มปุ๊บ ระบบจะขึ้นข้อความว่า "Triggering..." 
   - และถ้าสำเร็จ มันจะส่งอีเมลแจ้งเตือนไปที่ Inbox ของคุณจริงๆ ทันทีครับ! (ถ้าลองแล้ว เช็คอีเมลตัวเองดูนะครับ)

---

## 💡 สคริปต์เด็ดเอาไว้ตอบกรรมการสัมภาษณ์
ถ้ากรรมการถามเรื่องประสบการณ์ Power Apps/Power Automate ตามใน Resume ให้งัดสคริปต์นี้ไปตอบได้เลยครับ:

> *"ผมมีมุมมองว่า Power BI ไม่ควรเป็นแค่เครื่องมือดูข้อมูลแบบ Read-only ครับ แต่ควรจะเป็น **Actionable Dashboard** ได้ด้วย ผมจึงนำประสบการณ์จากการทำ Power Automate มาประยุกต์ใช้ โดยการฝังปุ่ม Flow ลงไปใน Power BI โดยตรงเลยครับ ตัวอย่างเช่น ถ้า User วิเคราะห์ข้อมูลแล้วเห็นว่ายอดขายตก เขาสามารถกดปุ่มเดียวบน Dashboard เพื่อดึง Filter context ในขณะนั้น แล้วสั่งยิงอีเมลส่งแจ้งเตือนไปหาผู้จัดการภาคได้ทันทีโดยไม่ต้องสลับหน้าจอไปมาครับ"*

ถ้าทำได้แบบนี้ รับรองว่าเหนือกว่าคู่แข่งทุกคนแน่นอนครับ! ลองทำตามดูนะครับ
